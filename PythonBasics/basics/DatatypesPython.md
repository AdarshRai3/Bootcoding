#Datatype Object
In Python we have two types of data types: mutable and immutable.

##Mutable data types(that can be changed inplace in the memory):
1> list
2> set
3> dict
4> bytearray
5> array
6> collections.deque

##Immutable data types(that cannot be changed inplace in the memory):
1> int
2> float
3> bool
4> str
5> tuple
6> frozenset
7> bytes
8> NoneType
